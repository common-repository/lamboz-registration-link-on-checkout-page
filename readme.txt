== Description ==

"Lamboz Registration Link On Checkout Page" is a plugin which is used to add custom registration link on WooCommerce checkout page. Its required woocommerce plugin to work.

It's a woocommerce add-on which helps you to add any custom registration link on the checkout page.

It will provide you with the option to add,
1) Custom registration link
2) Notice Label
3) Registration link label

Please deactivate it if you are not using woocommerce plugin.

== Installation ==

1) Upload lamboz-custom-registration-link-on-checkout-page to the /wp-content/plugins/
the directory or install from plugin page.
2) Activate the plugin through the Plugins menu item in the WordPress Dashboard.
3) Once the plugin is activated, go to the Settings and click on "Lamboz Custom Registration Link"
4) You will see "Lamboz Checkout Page Settings" page where you can add "Custom registration link", "Notice Label", "Registration link label".
